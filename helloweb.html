Introduction to Web Programming

What is Web Programming?

Web programming involves creating websites and web applications that can be accessed over the internet using a web browser.

Front-End vs. Back-End

Web programming is divided into two main parts: front-end and back-end.

  • Front-End: Also known as client-side programming, it involves building the user interface and interaction elements that users see and interact with in their web browsers.
  • Back-End: Also known as server-side programming, it involves handling the behind-the-scenes logic, database operations, and server communication necessary to make a website or web application work.

HTML: The Structure of the Web

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It provides the structure and content of a webpage.

HTML consists of a series of elements, each surrounded by angle brackets, which define the structure of a webpage. Elements can include headings, paragraphs, images, links, and more.

Below is a simple example of HTML code:

            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>My First Web Page</title>
            </head>
            <body>
                <h1>Hello, World!</h1>
                <p>This is my first webpage.</p>
            </body>
            </html>

HTML elements are nested within each other, forming a hierarchy that defines the structure of the webpage.

CSS: Adding Style to Web Pages

CSS (Cascading Style Sheets) is used to add styles and layout to web pages created with HTML. It allows you to control the appearance of text, images, and other elements on a webpage.

CSS works by selecting HTML elements and applying styles to them. Styles can include colors, fonts, margins, padding, and more.

Below is an example of CSS code that styles the <h1> element with blue color and centered alignment:

            h1 {
                color: blue;
                text-align: center;
            }

To apply styles to HTML elements, you can use selectors such as element names, class names, or IDs.

Getting Started

To get started with web programming, you can create HTML files using a simple text editor like Notepad (Windows) or TextEdit (Mac). Save the file with a .html extension, and open it in a web browser to see your webpage in action.

Next Steps

Once you're comfortable with HTML, you can learn CSS to add styles and layout to your web pages, and JavaScript to add interactivity and dynamic behavior.
